Condicionais

Condicionais são formas do código realizar determinada ação, dependendo do valor booleano de uma expressão. Declara-se uma condicional com as palavras-chave if... e else... , onde, caso determinada expressão seja True , o código abaixo do if será realizado e, caso a expressão tenha valor False , será validado o código dentro do else .

if condition: action1()
else: action2()

Também podemos comparar diversas condições, usando vários elif... (else if) em sequência:

if condition1: action1()
elif condition2 action2()
elif condition3 action3()
else: action4()

Vale notar que caso a condition2 seja considerada verdadeira, a checagem da condition3 não ocorrerá, tal que a ação abaixo dela não ocorra, mesmo se a condição seja verdadeira.
